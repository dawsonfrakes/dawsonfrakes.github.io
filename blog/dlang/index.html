I recently tried out Jai and it made me realize how great D already is. It also made me realize Zig is *very* D-like.

Expressivity

While D can be as expressive as Python with its UFCS, Slices, Dynamic Arrays, Associative Arrays, Lambdas, Delegates (ew), Object Oriented (ew), Concurrency (ew), Garbage Collection (ew), etc... minimalism is more on track with what I want. We can do better than what the standard library implements because we know our specific use cases. Not to mention you *must* implement your own code if you want it to work with -betterC.

"dmd -betterC -run main".split.spawnProcess.wait;
